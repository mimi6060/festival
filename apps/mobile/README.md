# Festival Mobile Application

Placeholder pour l'application React Native (iOS & Android).

## Setup

Pour initialiser cette application React Native, exécutez:

```bash
npx nx generate @nx/react-native:application --name=mobile --directory=apps/mobile
```

## Features prévues

- Programme des événements (offline-first)
- Notifications push
- QR Code billetterie
- Carte interactive avec géolocalisation
- Mode hors-ligne
