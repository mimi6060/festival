# Festival Web Application

Placeholder pour l'application Next.js principale (frontend public).

## Setup

Pour initialiser cette application Next.js, exécutez:

```bash
npx nx generate @nx/next:application --name=web --directory=apps/web --style=scss
```

## Features prévues

- Page d'accueil du festival
- Programme des événements
- Billetterie
- Carte interactive
- Espace utilisateur
