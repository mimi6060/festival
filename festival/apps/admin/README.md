# Festival Admin Application

Placeholder pour l'application Next.js d'administration (backoffice).

## Setup

Pour initialiser cette application Next.js, exécutez:

```bash
npx nx generate @nx/next:application --name=admin --directory=apps/admin --style=scss
```

## Features prévues

- Dashboard analytique
- Gestion des événements
- Gestion des artistes
- Gestion des utilisateurs
- Configuration des scènes
- Rapports et statistiques
- Gestion des ventes
